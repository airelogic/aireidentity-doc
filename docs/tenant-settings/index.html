Tenant Settings â€‹

Once a tenant gets created, a tenant admin can furnish/updates some information pertaining to the tenant. They are as follows:

  • Name: The name of the tenant that gets displayed in the dashboard
  • Support Email: The email address that users can contact for support
  • Tenant Logo Url: The URL of the logo that gets displayed to the users
  • Enable MFA for Login: Enable/Disable Multi-Factor Authentication for the tenant

When MFA enabled for a tenant , it's users will be prompted to set up two-factor authentication the next time they log in with that tenant.

Tenant users now have the flexibility to choose between two secure methods for verifying their identity when logging in:

  • Authenticator App: Use an authentication app (such as Google Authenticator or Authy) to generate a time-based, one-time 6-digit code.
  • Email Verification: Users receive a 6-digit verification code to their registered email address to complete the login process.

When new tenants are created, MFA Login is enabled by default.
