Tenant Providers ​

Providers allow you to define how users are able to log in to your tenant.

Overview ​

This page explains how to manage providers in AireIdentity. Identity providers are used to authenticate users in your tenant. You can add and configure which identity providers are allowed for your users when logging in.

The various options are explained in the following sections.
